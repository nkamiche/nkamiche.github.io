PROJECT DESCRIPTION

For my final project, I created Nice Kangaroo Care Kits, a multi-page website for a fictional business that offers customizable care packages for students, friends, family members, and special occasions. The business uses my Nice Kangaroo mascot and the tagline Pouched with purpose.

PAGES

The website includes these pages:
- index.html (Home)
- about.html (About)
- products.html (Products)
- services.html (Services)
- hours.html (Hours)
- contact.html (Contact)

Each page uses the same shared header and footer component files, includes original written content, and includes at least one relevant image with descriptive alt text.

DYNAMIC JAVASCRIPT FEATURES

1. Customizable care-kit builder (products.html)
   Users select kit items with checkboxes, and JavaScript displays a live estimated total and selected-item summary without reloading the page.

2. Interactive product slideshow (index.html)
   JavaScript powers a featured care-kit slideshow with Previous and Next buttons, updated image alt text, captions, and status feedback.

3. Order scheduling form (contact.html)
   Users submit pickup details through a form, and JavaScript generates a personalized confirmation summary on the same page without reloading.
